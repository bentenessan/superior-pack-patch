Files here are copied ADDITIVELY into each client's  config/  folder on update
(overwrite matching files, never delete). Use ONLY for CLIENT-side config files
that must match the pack. Mirror the layout under config/, e.g.:
   config-overlay/somemod-client.toml   ->   config/somemod-client.toml

Do NOT put per-user settings here (keybinds, sound, video) - those are personal.
Normally this folder is empty (just this note).
